# Markdown

副檔名為.md

## 常見語法
  * ### 標題 `#` `##` `###` `####` `#####`
  * 列點 `*` `-`
  
    ```markdown
    # 有階層關係
    - 大點
      - 小點
        - 小小點
    ```

  * **粗體** `**...**`、*斜體* `*...*`、~~刪除線~~ `~~...~~`
  * [連結]() `[標題](https://...)`
  * ![圖片]() `![圖片名](https://...)`
  * 分隔線  `___`
  * `程式區塊`
  
    ```markdown
    　`小區塊`

    　```SQL
    　大區塊
    　```
    ```

  * |表格|
    |--|
 
    ```markdown
    |標題欄|標題欄|
    |--|--|
    |第一列|第一列|
    ...
      ```

## 應用方式

* 使用Visual studio code進行編輯、預覽
* 使用Github搭配版控，Push .md檔案，可直接於網路上展示
* 使用Markdown to HTML工具，可顯示於網頁 (如部落格)
