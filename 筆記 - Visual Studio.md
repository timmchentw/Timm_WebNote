## Shortcuts

### Display related

* `ctrl+-` `ctrl+shift+-` 上/下一個檢視紀錄
* `ctrl+f` `ctrl+shift+t` 搜尋/全域搜尋
* `ctrl+,` `ctrl+t` Function & Property搜尋
* `ctrl+shift+space` 顯示Input Description
* `alt+2` 顯示Reference
* `ctrl+alt+pageup` `ctrl+alt+pagedown` 上/下一個Tab

### Edit related

* `ctrl+w` 選取單字
* `ctrl+shift+l` 刪除單行
* `ctrl+x` 剪下單行
* `ctrl+c` 複製單行
* `ctrl+v` 貼上單行
