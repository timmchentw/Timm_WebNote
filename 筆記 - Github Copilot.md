* [Copilot for Docs](https://githubnext.com/projects/copilot-for-docs/)
  * 依據程度搜尋文件

* Copilot Inline
  * 開啟檔案可以參考，讓功能更聰明
  * Comment提供的資訊更多，產出的程式碼會更精準
  * 自動在類似的地方加入建議 (比如註解、產生測試)
  * 重複的Code適合使用

* Copilot Chat
  * VS也可以使用
  * 可用於重構程式碼
  * 給予更多語法與優化方法
